Один из учебных проектов, сделанный в рамках курса от GloAcademy.
Данный проект полностью сделан мной самостоятельно примерно за неделю, работал с макетом в Figma.
В данном проекте использовал:
1) Методология БЭМ
2) Препроцессоры SCSS
3) GULP
4) частично CSS-анимации
5) JS-библиотеки: Swiper(слайдер), AOS(анимации)
6) Рабочая форма c использованием PHPmailer
